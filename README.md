# sysu-competition-system

## 中山大学计算机竞赛系统后端
### 技术栈：React+Antd+Node.js+MongoDB
### 客户端：https://github.com/H-JW0829/sysu-competition-system-client  
### 管理端：https://github.com/H-JW0829/sysu-competition-system-admin  
### 功能
1.用户管理  
2.作品管理  
3.评论管理  
4.竞赛管理  
### 启动命令
需要先在你的电脑上配置好mongodb  
npm install  
npm run serve
